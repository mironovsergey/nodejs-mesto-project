# Mesto API

## Установка и запуск

Установка зависимостей

    npm install

Запуск с авторестартом

    npm run dev

Запуск без авторестарта

    npm start

Сборка билда

    npm run build

Запуск линтера

    npm run lint

## Переменные окружения `.env`

```
JWT_SECRET=
MONGO_URI=mongodb://localhost:27017/mestodb
PORT=3000
```
